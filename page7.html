Created in P2 Map Editor

 

Constraints

Create a level with a 2/5  difficulty

 

Walkthrough Duration

21 hours

PreviousNextClose





Light's Up is a puzzle level developped at the Campus ADN with the Portal 2 Map Editor.

The players only have one room and one objective: escape while using mechanics seen in the main game.

The puzzle was designed to be easy for players with good observational skills. The main challenge is to get to the second floor, the way to achieve this is rather unique.

Design Intentions

Observation

This level only has one room, there are no surprises to find elsewhere. Everything the players need to complete this challenge can be seen from the very beginning. The main goal of this design is to ensure that the players always know where they are and can notice what other ingredients are available at all times.

Reward Ingenuity

The main surprise when resolving this puzzle is to understand that you can use the Light Bridge with an Angled Panel to create a 45° walkway. Once you get to the 2nd floor, it's quite easy to complete the rest of the level. The players can then have fun with the Flinging since there is great verticality in the room.

Visibility

It was important for me to let the players see everything in this level. A lot of Glass Panels are used for this purpose but also as a gameplay mechanic. The Lasers can go throught these panels which are needed to activate the Relays. 

Teaching Mechanics

The main challenge of this puzzle is to know all of the mechanics of Portal2, this is since it's suppose to be a DLC style level. Many of my colleagues at the Campus ADN didn't know all of them. I created a smaller first room to teach them basic information needed to complete my main puzzle such as the Light Bridge which can pass through portals.
